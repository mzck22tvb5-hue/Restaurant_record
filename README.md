# 我的餐廳地圖（行動版 / iPhone）
- 浮動按鈕：新增/搜尋/匯出/匯入
- 底部面板：編輯名稱、評分、標籤、備註、照片
- Google 文字搜尋批次加入「想去」
- 地址定位加點
- 匯入/匯出 JSON
- PWA：可加入主畫面（注意：Google 地圖需連網）

## 使用
1. 打開 `index.html`，把 `YOUR_GOOGLE_MAPS_API_KEY` 換成你的金鑰（需啟用 Maps JavaScript API + Places API）。
2. 將整個資料夾部署到 GitHub Pages / Netlify / 任何靜態空間。
3. 用 iPhone Safari 打開網址 → 分享 → 加入主畫面。
